        TASK 4 - DATA VISUALIZATION

Internship Program : Decoded Labs Virtual Internship
Domain             : Data Science
Project            : Project 2 - Task 4
Task Name          : Data Visualization

----------------------------------------
OBJECTIVE
----------------------------------------

The objective of this task is to create visual
representations of employee data using Python
libraries such as Matplotlib and Seaborn.

The visualizations help in understanding:
- Employee distribution
- Salary patterns
- Gender ratio
- Experience analysis
- Department insights

----------------------------------------
TOOLS & TECHNOLOGIES USED
----------------------------------------

- Python
- VS Code
- Pandas
- Matplotlib
- Seaborn

----------------------------------------
FILES INCLUDED
----------------------------------------

1. employees_data.csv
   -> Dataset containing employee information

2. visualization.py
   -> Python script for generating charts and graphs

3. README.txt
   -> Project documentation file

----------------------------------------
VISUALIZATIONS CREATED
----------------------------------------

1. Department-wise Employee Count
   - Bar Chart

2. Gender Distribution
   - Pie Chart

3. Salary Distribution
   - Histogram

4. Experience vs Salary
   - Scatter Plot

5. Average Salary by Department
   - Bar Graph

----------------------------------------
HOW TO RUN THE PROJECT
----------------------------------------

Step 1:
Install required libraries using terminal:

pip install pandas matplotlib seaborn

Step 2:
Place the CSV file and Python file in the same folder.

Step 3:
Run the Python file:

python visualization.py

----------------------------------------
OUTPUT
----------------------------------------

The program generates multiple graphs and charts
that help analyze employee data effectively.

----------------------------------------
CONCLUSION
----------------------------------------

This project demonstrates the use of data
visualization techniques to communicate insights
from employee datasets clearly and effectively.
